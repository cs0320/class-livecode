# React + TypeScript + Vite

This is a simple React web application echoing the "plain" TypeScript application. This was generated via prompting in class, so **don't take it as a perfect template**! The settings, etc. _work_ but they may not be ideal for your needs. 

This project might look more complex and annoying than the plain TypeScript version. It's true that there's a lot more configuration and code. But most of that is boilerplate, and as applications get more complicated the structure Reacts adds is valuable. 

To run this, run `npm run dev`
